Nfv Deployer RESTful Web Service

The Nfv Deployer web service is a simple web service that allow its client to:

  • Read all the information available in DP2-NFV about NF-FGs, hosts and
    • reachable hosts
  • Deploy a new NF-FG into the system
  • Undeploy an NF-FG
  • Add or remove a single node in an already deployed NF-FG
  • Add or remove a single link in an already deployed NF-FG
